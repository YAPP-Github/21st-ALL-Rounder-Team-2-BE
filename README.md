## 1. 주요 스택

- spring boot:2.7.6
- jdk:11
- mysql:8.0

## 2. 커밋 메시지

커밋 메시지 형식은 아래와 같습니다. `subject`까지만 작성해도 괜찮습니다.

```
<type>(<scope>): <subject>
<BLANK LINE>
<body>
<BLANK LINE>
<footer>
```

## 2.1 커밋 타입(`<Type>`) 
- `feat` :  (feature)
- `fix` :  (bug fix)
- `docs`:  (documentation)
- `style` : (formatting, missing semi colons, …)
- `refactor`
- `test` : (when adding missing tests)
- `chore` : (maintain)

## 3. 패키지 구조

- `module`
    - `controller`
    - `domain`
    - `dto`
    - `exception`
    - `repository`
    - `service`

## 4. 인텔리제이 자동 재시작 설정

1. 설정  >  빌드, 실행, 배포  >  컴파일러  > 프로젝트 자동 빌드 체크
2. 고급 설정 > 컴파일러 > 개발된 애플리케이션이 현재 실행 중인 경우에도 auto-make가 시작되도록 허용 체크
